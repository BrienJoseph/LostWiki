2004Scape Ported TypeScript Client Wiki

Source Repository

This section of the wiki documents the 2004Scape Ported TypeScript Client. This client is a modern TypeScript port of the original 2004Scape Java client.

Source: 2004scape/Client2

Important Note

This client was fully ported by the 2004Scape development team. There was no source code leak from Jagex. All code in this repository was manually recreated from decompiled and analyzed client behavior, respecting copyright boundaries.

Mission Statement

The mission of this documentation is to explain the structure and functionality of the TypeScript client, assist developers in navigating the codebase, and highlight the tools and features available within the client.

Features

  • Playable web client hosted on GitHub Pages
  • Developer Playground environment
  • Item Viewer (browse and inspect in-game items)
  • Message Animation Viewer (visualize text effects)
  • Sounds Viewer (play in-game audio)
  • Cache Viewer (Work In Progress)
  • Interface Editor (Work In Progress)

Goals

  • Document client structure and design decisions.
  • Explain how the TypeScript systems correspond to the original Java client.
  • Guide new contributors through setup, tooling, and feature development.
  • Showcase the development tools provided in the client for asset inspection and debugging.
