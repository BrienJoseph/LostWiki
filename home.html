About the project

This is the unofficial wiki for 2004scape, a project created by Pazaz and LostCity. This is a community driven, open source attempt to document accurately per the code to keep accurate information for each period of time the Lostcity project will host.

The difference, in terms of vision, is to allow this Wiki to become a living document that works hand-in-hand with the 2004scape project. With a focus on historical accuracy and source code, all pages should be based on the content from which the preservation project has been derived from. The other huge target for this wiki is that all drop rates and other mechanics are not assumptions or heresay, we can directly reference the code to verifiably confirm these components of the game.
